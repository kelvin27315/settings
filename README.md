# settings
さまざまな設定ファイルの置き場所

## Zsh
設定ファイルの置き場所

`$HOME/.zshrc`

## Alacritty
https://github.com/alacritty/alacritty/

設定ファイルの置き場所

`$HOME/.config/alacritty/alacritty.toml`

## WezTerm
https://wezfurlong.org/wezterm/

設定ファイルの置き場所

`$HOME/.config/wezterm/wezterm.lua`

## Starship
https://starship.rs/ja-JP/

設定ファイルの置き場所

`$HOME/.config/starship.toml`
